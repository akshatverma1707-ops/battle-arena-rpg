# Battle Arena RPG ⚔️

A turn-based RPG battle game written in **C++** demonstrating Object-Oriented Programming concepts.

## Features

- Character selection (Knight, Wizard, Archer, Assassin)
- Turn-based battle system
- Health bars
- XP progression system
- Critical hits
- Potion healing
- Enemy taunts
- Final boss battle (Dragon)

## OOP Concepts Used

- Inheritance
- Polymorphism
- Encapsulation
- Class interaction

## How to Run

Compile the program using g++:

g++ src/main.cpp -o game

Run the game:

./game
